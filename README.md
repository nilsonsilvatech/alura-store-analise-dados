Alura Store – Análise de Desempenho para Tomada de Decisão

Contexto do Problema

O Sr. João possui quatro lojas da rede Alura Store e deseja vender uma delas para investir em um novo negócio.

O objetivo deste projeto é analisar os dados de vendas e desempenho das lojas para identificar qual apresenta menor eficiência e, com base em dados, recomendar qual deve ser vendida.

Métricas Analisadas

Faturamento total por loja
Categorias mais vendidas
Média de avaliação dos clientes
Produtos mais e menos vendidos
Custo médio de frete

Tecnologias Utilizadas

Python
Pandas
Matplotlib

Google Colab

Objetivo da Análise

Gerar insights estratégicos baseados em dados para apoiar uma decisão de negócio.
